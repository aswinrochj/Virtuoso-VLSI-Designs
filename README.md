# 🧠 VLSI Design Portfolio (Cadence Virtuoso)

##  Overview
This repository contains my work in **VLSI design using Cadence Virtuoso**.  
It includes various CMOS circuit designs developed as part of learning and hands-on practice.

---

##  Design Flow
The designs in this repository follow the standard VLSI design flow:
- Symbol Design  
- Schematic Design  
- Layout Design  

---

##  Tools & Technology
- Cadence Virtuoso  
- gpdk090 Technology  
- CMOS Design Principles  

---

##  Repository Structure
Each design is organized into its own folder.  
Every folder contains:
- Design images (Symbol, Schematic, Layout)  
- A README file describing the implementation  

---

##  Objective
To build a strong foundation in **VLSI design and CMOS circuit implementation**, focusing on translating theoretical concepts into practical layouts.

---

##  Future Scope
- Addition of more complex digital and analog circuits  
- Exploration of advanced VLSI design techniques
